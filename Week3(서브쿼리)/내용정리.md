# 서브쿼리
## WHERE 절 서브쿼리
### 단일행 서브쿼리
비교 연산자와 함께 사용
```sql
SELECT *
FROM tips
WHERE total_bill > (SELECT AVG(total_bill) FROM tips) -- 서브쿼리의 결과값은 1개
```
### 다중행 서브쿼리
IN, NOT IN과 함께 씀
```sql
SELECT *
FROM tips
WHERE total_bill > (SELECT AVG(total_bill) FROM tips) -- 서브쿼리의 결과값은 1개
```
### 다중컬럼 서브쿼리
```sql
SELECT *
FROM tips
WHERE (day, total_bill) IN (
  SELECT day, MAX(total_bill)
  FROM tips
  GROUP BY day
) -- 서브쿼리의 결과값은 컬럼 N개, 로우 N개
```
